# Community CarPool Tool — Squad Team

## Project Context
- **Project:** Community CarPool Tool
- **User:** Kusumakar Althi
- **Stack:** React + Vite + Tailwind CSS, Node.js + Express, PostgreSQL, Prisma, JWT + bcrypt
- **Description:** A community carpooling platform where drivers schedule trips, riders browse and request to join, and credits are calculated based on distance using the Haversine formula.
